# Шорт-Лист Мифов Для Главного Hero

Этот документ фиксирует отдельную редакционную выборку мифов для главного hero. Это не копия старой шкалы `A/B/C`, а новая линейная переоценка после просмотра всех формулировок `Итоговая` подряд.

Во всех production-вариантах hero перед формулировкой показывается отдельный UI-лейбл `Заблуждение`. Сам текст мифа хранится и документируется без этого интерфейсного префикса.

## Принцип Отбора

В shortlist попадают мифы, которые одновременно:

- быстро считываются в hero без длинного контекста;
- цепляют широкую аудиторию, а не только узкую нишу;
- хорошо звучат как первый экран и не разваливаются без дополнительного пояснения;
- могут вести либо в страницу темы, либо в конкретное событие без смыслового провала;
- не выглядят как локальная справка, архивная мелочь или техническая trivia.

Отдельно учитывается разнообразие: даже сильные мифы нельзя собирать подряд в один тональный кластер `уничтожили / разрушили / ничего не осталось`.

## Sharp Reserve

Это сильные мифы, но их лучше не ставить основой первой версии hero-пула. Они либо стилистически более разговорные, либо тематически пересекаются с уже выбранными приоритетами.

| Миф | Тема | Событие | Почему лучше держать в резерве |
|---|---|---|---|
| Калининградская повседневность — это только лёгкость, милота и красивый вайб. | Быт, привычки и частная жизнь | Привычки калининградцев, юмор, суеверия не только подростковые, страшилки, легенды калининградских дворов | Современный и заметный, но стилистически более разговорный, чем основной пул. |
| Руководители советского Калининграда в основном только уничтожали немецкое наследие. | Город, архитектура и среда | Человек, заложивший фундамент современного Калининграда. Виктор Денисов и его эпоха | Сильный тезис, но тематически пересекается с несколькими более приоритетными мифами. |

## Не Брать В Главный Hero Без Дополнительной Переработки

- локальные trivia-мифы вроде `золотой клёпки`, `бегемота Ганса`, отдельных скульптур и курсантских баек;
- слишком нишевые или технические формулировки без широкого крючка;
- мифы, которые звучат как фактологическая справка, а не как сильное заблуждение;
- формулировки, которым нужен длинный контекст, чтобы не показаться случайным утверждением.

## Проблемы Source, Которые Лучше Закрыть До Публикации

- В блоке `Космическая орбита Калининграда` есть смысловой конфликт между `Исходник`, `Рекомендуемая формулировка` и `Итоговая` у мифа про промышленность.
- В блоке `Калининградское здравоохранение...` есть формулировка, которая пока звучит скорее как констатация проблемы, а не как миф.
- В source есть несколько явных орфографических и редакционных шероховатостей, которые не стоит тянуть в production hero без нормализации.

## Редакционные Ограничения Для Ротации

- Не ставить подряд больше одного мифа из кластера `уничтожили / разрушили / невозможно спасти`.
- В первых `10` hero-показах одной сессии не должно быть больше `2` мифов из кластера `уничтожили / разрушили / невозможно спасти`.
- Не ставить подряд два мифа из одного и того же события.
- На главной в рамках одного shuffle-цикла по возможности не показывать больше одного мифа из одного и того же события, если в пуле есть другие доступные варианты.
- Не делать первый myth-screen слишком обвинительным; первый экран должен быть сильным, но не отталкивающим.
- Для `v1` допустим основной пул порядка `12-18` мифов и небольшой резерв; тащить весь source в главный hero не нужно.
