- **Статус события:** подтверждено / дата будет объявлена позже / регистрация скоро откроется / мест нет.

- **Длительность и режим посещения:** отдельное поле, чтобы одинаково показывать разовые события и выставки.
- **Спикер / участники:** отдельная сущность с ФИО, короткой регалией, фото, должностью, ссылкой на все события спикера.
- **Короткий hook-текст:** 1–2 предложения для афиши и карточки события.
- **Полное редакционное описание:** драфт, из которого потом можно делать новости, релизы и посты.
- **3 вопроса / 3 мифа:** отличные блоки для тизеров, карточек регистрации и промо в соцсетях.
- **Цитата спикера о лекции** и **цитата о фестивале:** их удобно использовать как врезки, подписи и промо-цитаты.
- **Формат участия:** нужна ли регистрация, свободный вход, рекомендуемый возраст, доступность для семейной аудитории, возможность вопросов из зала.
- **Партнёр / площадка:** отдельное поле для логотипов, страниц площадок и фильтрации программы по локациям.
- **Тематические теги:** море / город / архитектура / память / экология / искусство / повседневность / переселенцы / наука.
