# Группировка программы фестиваля для сайта / лендинга

## Рекомендация по названию блока
- **Название в верхней навигации:** `По интересам`
- **Название блока / шапки на лендинге:** `Маршруты по интересам`

Такой вариант остаётся самым понятным для пользователя: в навигации он короткий и ясный, а на странице позволяет показать не жёсткие рубрики, а пересекающиеся подборки событий.

## Что изменено в этой версии
Паблик-токи **не только сохранены внутри тематических маршрутов**, но и **вынесены в отдельную группу**. Это даёт пользователю ещё один понятный вход в программу: не по теме, а по типу опыта — для тех, кому важны живой разговор, обмен мнениями, возможность задавать вопросы и слышать несколько позиций в одном событии.

Ниже — **перекрывающиеся группы**, а не взаимоисключающие разделы.
